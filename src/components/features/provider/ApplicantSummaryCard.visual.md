## Responsive Behavior

### Desktop (â‰¥ 1024px)
- Stats grid: 2 columns (2x2)
- Full component width: ~400-500px (recommended max-width)
- Sticky positioning when in sidebar

### Mobile (< 768px)
- Stats grid: Stacks to single column if needed
- All text remains readable
- Touch targets: 44px minimum

## Component Sizing

### Recommended Dimensions
- **Width**: 100% (max-width: 400-500px recommended)
- **Height**: Auto (varies based on content)
  - Minimal (first-time client, no insights): ~400px
  - With history + insights: ~600px
  - Fully expanded (many sessions): ~800px

### Spacing
- **Card padding**: `p-6` (24px)
- **Section gaps**: `space-y-4` (16px)
- **Stats grid gap**: `gap-3` (12px)
- **Individual stat padding**: `p-3` (12px)
