## Color Coding

- **Yellow/Primary (#F7E547)**: Commission breakdown card background
- **Green (#10B981)**: Connected state, success indicators
- **Yellow/Amber (#FBBF24)**: Pending state, warnings
- **Gray (#6B7280)**: Not connected state, neutral elements
- **Orange (#F97316)**: Tax information alert
- **Blue (#3B82F6)**: Info alerts when skipping

## Key Elements

1. **Commission Breakdown** - Always visible, top priority
2. **Stripe Connection Status** - Changes based on state
3. **Payout Information** - Static, always shown
4. **Tax Alert** - Static warning, always shown
5. **Navigation** - Adaptive based on connection status
6. **Skip Warning** - Only shown when not connected
